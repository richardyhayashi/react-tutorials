## Notes

## Create React Project

`$ npm create vite@latest react-crash-2024` Choose: React, JavaScript<br>
`$ cd react-crash-2024`<br>
`$ npm {i|install}`

### Set Port

In `vite.config.js`:<br>
`export default defineConfig({`<br>
`  ...`<br>
`  server: {`<br>
`    host: "0.0.0.0",`<br>
`    port: 3000,`<br>
`  },`<br>
`  ...`<br>
`})`

### Install Tailwindcss

'https://tailwindcss.com/docs/guides/vite'

### Run Server

`$ npm run dev`
