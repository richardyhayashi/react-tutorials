    `$ npmx json-server -p 3500 -w data/db.json`

## Create React Project

`$ npx create-react-app {.|react-full-course-gray}`

### Run React App

`$ npm start`

### Install React Icons

`$ npm i react-icons`

`$ npx json-server -p 3500 -w data/db.json`
